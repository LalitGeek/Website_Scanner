# How to Use Webpage Scanner

Follow these steps to get the most out of the Webpage Scanner extension:

### 1. Launching the Extension
- Navigate to any website you wish to analyze.
- Click on the **Webpage Scanner** icon in your Chrome toolbar.
- The popup interface will appear and automatically begin a basic analysis of the current page.

### 2. Performing a Scan
- If the scan doesn't start automatically, or if you want to refresh the data, click the **Scan Now** button in the popup.
- The status indicator at the top will show the progress of the scan.

### 3. Reviewing Results
The results are organized into several tabs/sections:
- **Technologies**: View the detected frameworks, servers, and libraries with confidence percentages.
- **Security**: Check if the site passes essential security header checks and HTTPS requirements.
- **Insights/SEO**: Review metadata, heading structures, and asset counts.
- **Domain Intel**: See the server's IP address, network provider, and OS.
- **Contacts**: View discovered email addresses and phone numbers.

### 4. Exporting Data
- **PDF Report**: Click the "Export PDF" button to generate a professional, enterprise-style audit report. This includes a screenshot of the site (if enabled).
- **JSON Data**: Click "Export JSON" to download the raw data for further analysis.
- **Copy to Clipboard**: Use the "Copy" icon to quickly copy the JSON result to your clipboard.

### 5. Troubleshooting
- **No data found**: Ensure you are on a valid webpage (not a Chrome internal page like chrome://settings).
- **PDF Export fails**: Ensure the page has finished loading before clicking export, as it may need to capture a screenshot.
- **Permissions**: Ensure you have granted the extension permission to access the site if prompted.
