# How to Use Scanify | Web Auditor

Follow these instructions to run and evaluate your audited pages using Scanify:

### 1. Launching the Auditor
- Open your browser and go to the target website you wish to scan.
- Click on the **Scanify | Web Auditor** logo in your Chrome toolbar.
- The glassmorphic SaaS dashboard will display. The scanner will automatically load pre-cached results or launch a live analysis of the active page.

### 2. Navigating the Dashboard Sections
Scanify distributes complex audits across 5 premium panels:
- **Overview**: 
  - **Audit Score**: A radial score (0-100) combining Security configuration and SEO validity.
  - **SaaS Indicators**: Fast diagnostics for Load Time (in ms) and total HTML script/image assets.
  - **Security Audits**: Real-time checklist for HTTPS, CSP, HSTS, Secure Cookies, and Mixed Content.
  - **Key Indicators**: Fast pills for WebSockets, Server OS, Sitemap, and fonts discovered.
  - **Media Assets Downloader**: Displays the total count of deduplicated media files (images, SVGs, audio, video, picture source files) scanned at the highest available resolutions. Features a **Download All Media** button that delegates a rate-limited queue to the persistent background service worker, allowing the batch download to complete seamlessly even if you close the extension popup. Downloads are neatly sorted into `Scanify_Media/[domain-name]/`.
- **Stack (Tech Stack)**:
  - Lists frameworks, analytics engines, web servers, and CDNs grouped by tier.
  - **Search filter**: Use the input box to instantly filter technologies by category or name.
- **Intel (Domain Intelligence)**:
  - Network metadata (IP, ASN, ISP provider, server host country, OS).
  - **Discovered Subdomains**: Lists active subdomains resolved via background passive DNS querying.
  - **DNS Zone Records**: Scrollable grid showing A, AAAA, MX, NS, and TXT records.
- **SEO (SEO & Contacts)**:
  - Search Optimization metadata (Page Title, Meta Description, Canonical URL).
  - Headings pill count (H1, H2, H3 tags).
  - **Extracted Contacts**: Formatted emails and phone numbers discovered in the page content.
- **Console (Dev & Storage)**:
  - **PWA Status**: Checks Web Manifest file presence, Service Worker activity, and PWA capability status.
  - **JS Insights**: Scans for loaded framework libraries (React, AngularJS, Vue, lodash, jQuery, HTMX, Alpine, etc.) and lists custom global variables declared on `window`.
  - **Client Storage**: Displays count of active Cookies, LocalStorage items, and SessionStorage keys.
  - **Storage Auditing**: Includes a detailed, scrollable Cookies grid checking Secure/HttpOnly security flags, plus badge lists of all Local/Session storage keys.

### 3. Header Action Controls
At the top right of the dashboard, you have:
- **Status Indicator**: Shows state (`Scanning`, `Scanned`, `Exporting`).
- **🔄 Rescan button**: Triggers a manual page re-evaluation.
- **📥 Export JSON button**: Downloads raw JSON structured audit results.
- **📋 Copy to Clipboard button**: Copies the full JSON block to your clip register.
- **💖 Donate & Support button (Heart icon)**: Opens a secure Razorpay checkout page in a new tab.

### 4. Professional PDF Exports
- Click **Generate Audit PDF** at the bottom of the popup.
- Scanify will capture a screenshot of the active tab, load the brand logo, assemble clean tabular layouts, and compile an executive audit report, saving it directly to your downloads.

### 5. Troubleshooting
- **Missing Domain Intel**: Ensure the site is public and you are connected to the network, as DNS/IP APIs query external servers.
- **Permissions**: If a site blocks scans, ensure the extension has "Allow access to all sites" checked in extension settings.
